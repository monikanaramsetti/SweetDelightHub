# Sweet Delights Bakery Website

A modern, responsive website for a bakery business featuring product showcase, shopping cart functionality, and order management.

## Features

- 🍰 Product Showcase
- 🛒 Interactive Shopping Cart
- 📱 Responsive Design
- 🛍️ Checkout System
- 📝 Contact Form
- 🎨 Modern UI/UX

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Responsive Design
- CSS Grid & Flexbox

## Project Structure

```
sweet-delights-bakery/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Getting Started

1. Clone the repository:
